This application is for D&D (Dungeons and Dragons) game.
User can create a party, join a party, choose role in the party and start the game.
There will be two type of role: 
1. Dungeon Master (DM)
2. Party member

Every party has only 1 DM and a minimum of 3 party members to start the game.

UI should be separated for DM and Party member.
DM can create a party, can see all party members.
Party member can see only DM and other party members (not able to see other party members information).

The game is chat-based. So there should be a chat interface where DM and Party members can chat with each other.

After party is ready, DM can start the game. And chat interface will change to game interface where DM can send the story to the party members and party members can respond to DM.
